# YumStudio Object Notation

YumStudio Object Notation (.yso files) is a simple object notation format used by the YumStudio game engine.

## Syntax

```yso
[section]
key: value
LongKey: """
damn
"""
smth: 789

[otherSection]
password: "12345678" ; Not secure btw!
                     # Also a comment!
```
